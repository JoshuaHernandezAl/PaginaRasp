¿Como iniciar en Raspberry Pi 4?

Para comenzar a trabajar con una Raspberry Pi (en particular con la Pi 4), primeramente debemos configurarla, desde cargar con raspbian (SO basado en debian para la raspberry pi) la Raspberry, hasta poder acceder a ella de manera remota y que sea precindible el uso de perifericos definidos para esta y poder manejar nuestra raspberry desde un computador externo (el personal o de trabajo).
¿Pero como hacemos eso? Es bastante simple y cortesia de un canal en youtube te lo agregare a un link a continuacion :D

Iniciemos con la raspberry


Ya con nuestra raspberry configurada y lista para usar ¿que sigue?
saber como conectar nustros gpio (General Purpose Input/Output, Entrada/Salida de Propósito General) para eso les comentare que existen tres maneras de contar nuestros pines del J8

  • BCM:Es un protocolo muy usado para referirse a los gpio segun su conexion con el SoC
  • Board:Es la enumeracion de los pines segun como estan en la tarjeta de la raspberry
  • wPi:Es un protocolo que se utiliza la distribucion de gpio utilizada por PiJ4
